# AccountDetect(https://datarisk8.com/)
## 1:针对海外Facebook、Instagram、WhatAPP、Reddit等各种社交账号等资源，进行风险检测的智能平台: 支持新/老/冻结、CK检测、注册时间查询等。
目前支持：
* FB：新/老/冻结、CK检测、注册时间查询等；
* INS：新/老/冻结/、账户信息、账密检测等。
* WhatsApp：新/老账号检测
* Reddit：新/老/冻结，以及用户发帖、评论等内容、注册时间，用户id、活跃社区等信息查询

## 2.An intelligent platform for risk detection of various accounts such as Facebook, Instagram, WhatsApp, Reddit, etc., supporting checks for new/old/frozen status, CK validation, registration time inquiry, etc. 
Supported Features:
* Facebook：new/old/frozen accounts, CK detection, registration time query, etc.;
* Instagram：new/old/frozen accounts, account information, account and password detection, etc.
* WhatsApp：new/old.
* Reddit：Search for information such as new/old/frozen accounts, user posts and comments, registration time, user ID, and active community activity.
